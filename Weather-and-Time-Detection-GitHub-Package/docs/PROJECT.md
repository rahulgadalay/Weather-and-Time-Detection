# Weather & Time Detection AI Agent

## Project Description

The Weather & Time Detection AI Agent is an intelligent assistant built using Google's Agent Development Kit (ADK) and the Gemini API. It demonstrates how an AI agent can understand user intent, invoke the appropriate tool, and return real-time information.

The project uses Google's tool-calling capabilities to dynamically decide whether a user's query requires weather information or current time information.

## Project Workflow

1. User enters a query.
2. Gemini understands the user's intent.
3. Google ADK chooses the correct tool.
4. The selected tool calls the appropriate API.
5. Data is processed.
6. A formatted response is returned.

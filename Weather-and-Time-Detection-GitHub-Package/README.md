# 🌦️ Weather and Time Detection

AI-powered Weather & Time Assistant built with **Google ADK**, **Gemini API**, and **Python**.

## Features
- Real-time weather lookup
- Current time lookup
- Google ADK tool calling
- Gemini-powered reasoning
- Modular architecture

## Tech Stack
- Python
- Google ADK
- Gemini API
- REST APIs

## Installation

```bash
pip install -r requirements.txt
```

Create `.env`

```env
GOOGLE_API_KEY=your_api_key
```

Run:

```bash
python agent.py
```
